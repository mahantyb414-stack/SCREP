🔑 IMPORTANT NOTES
================================================================================

1. MEMORY:
   - Render allocation: 2GB (Standard plan recommended)
   - Chromium needs ~80MB per context

2. FREE TIER:
   - Services spin down after 15 min inactivity
   - Upgrade to Standard ($7/month) for 24/7 uptime

3. RATE LIMITING:
   - Google Maps may rate limit aggressive scraping
   - Adjust JITTER values in scrap_main.py if needed

4. DATA STORAGE:
   - Data stored temporarily in /outputs on Render
   - For persistent storage, add PostgreSQL/MongoDB

5. GITHUB ACTIONS:
   - Workflow runs weekly by default
   - Manual trigger available in Actions tab
